Kiwi.Plugins.InAppPurchase Class

Defined in: src\in-app-purchase-2.0.0.js:1
Module: Plugins
Parent Module: Kiwi

The InAppPurchase plugin is designed to help make intergrating the CocoonJS IAP with Kiwi easier. When a game is created, a new manager will be added to the 'game' root which is a instanted version of this class, under the property 'inAppPurchase'. You can then access that object to make a purchase or fetch purchases from the store.
