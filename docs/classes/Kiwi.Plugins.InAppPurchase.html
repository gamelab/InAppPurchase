Kiwi.Plugins.InAppPurchase Class

Defined in: src\inAppPurchase-1.0.0.js:1
Module: Plugins
Parent Module: Kiwi

The InAppPurchase plugin is designed to help make intergrating in-app purchase's using CocoonJS with Kiwi easier. What happens is that when a game that use's this plugin is created, a new manager is added to the 'game' root which is a instanted version of this class, under the property 'inAppPurchase'. You can then access that object to make a purchase or fetch purchases from the store.
